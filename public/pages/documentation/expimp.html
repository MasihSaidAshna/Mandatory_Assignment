Export and Import

Documentation on exporting and importing in Node.js

In Node.js you can export and import modules to manage and maintain large-scale applications easier. By default Node.js is used with CommonJS's module system, which involves: module.exports; and require();. Instead of using CommonJS's module system, you can use ES which has the keywords "export" and "import". ES generally offers more features over CommonJS.

CommonJS

In the following examples is how you export modules with CommonJS: 

                    const myFunction = () => {
                    };
                    
                    module.exports = myFunction;    
                    // Or you can use:
                    exports.myFunction = myFunction;
In this code snippet we are exporting a single function. To then import this function you write: 
                    const myFunction = require('./myModule');
                    // OR
                    const { myFunction } = require('./myModule');

ES6

Importing and exporting in ES is written differently than CommonJS. You export a default function this way: 

                    export default myFunction;
and importing a default export: 
                    import myFunction from './myModule';
